# Sage — Voice Mental Health Companion

Sage is a real-time, voice-interactive emotional support companion designed to provide a supportive hearing ear. By combining speech recognition, emotion detection, and advanced language modeling, Sage understands the emotional context of your words and responds with empathy and care using a natural-sounding voice.

## Setup Instructions

1.  **Install Dependencies**:
    Ensure you have Python installed, then run:
    ```bash
    pip install -r requirements.txt
    ```
2.  **Configure Environment**:
    Copy the `.env.example` file to a new file named `.env`:
    ```bash
    copy .env.example .env
    ```
    Open `.env` and add your **Gemini API key** next to `GEMINI_API_KEY=`.
3.  **Run the Application**:
    Simply double-click `run.bat` or run it from your terminal:
    ```bash
    run.bat
    ```

## Note: Ollama
Ollama is no longer required for this project. Sage now uses the Gemini API for more robust and empathetic response generation.
